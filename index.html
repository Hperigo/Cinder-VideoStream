Welcome to Cinder-VideoStream.

This Cinder block allows creating videostream server and client applications using asio framework in libcinder creative toolkit

How to use it

Download this project as a zip, or clone if from the github and place into Cinder Blocks directory. Launch Cinder's TinderBox app and, while creating your streaming application, select this block from the list of available Cinder blocks. A minimal application will be created automatically. It will use the default camera on your device. Open created c++ project and start customizing it, for example, replace the default camera with your camera or with any other video source, add UI controls. That's it!

Dependencies

Currently, this block is based on Cinder 0.9 release (aka glNext branch).
